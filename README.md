# lifecount - 人生一格

## 简介

本项目是一个参考了小程序 lifecount 而制作的网页版本，关于人生格式化，恰如名人莎士比亚说过：“人的一生是短的，但如果卑劣地过这一生，就太长了”。自由点，人生不应该被定义。

## 环境依赖

- Node.js：v18 及以上版本
- pnpm.15.5 及以上版本
- 浏览器：Chrome、Firefox、Safari 等现代浏览器

## 安装步骤

1. 克隆项目代码到本地

```bash
git clone https://github.com/hankliu62/lifecount.git
```

2. 进入项目目录

```bash
cd lifecount
```

3. 安装项目依赖

```bash
pnpm install
```

4. 启动项目

```bash
pnpm run dev
```

## 使用方法

访问 `http://localhost:3000` 即可看到人生一格的首页。

在首页上可以选择需要的工具进行使用。

工具的具体使用方法和参数可以参考每个工具的文档说明。
